# Analytics Export Module Report

The Analytics Export Module is responsible for visualizing processed stock market data using Pandas, Matplotlib, and Seaborn. The module reads data from `featured_data.csv` and generates multiple charts to analyze stock performance trends.

The module includes four main visualizations:

# Closing Price Line Chart
shows how stock closing prices change over time.
# Opening Price Line Chart
displays opening price trends across dates.
# Trading Volume Bar Chart
visualizes market trading activity and volume fluctuations.
# Correlation Heatmap
 identifies relationships between numerical stock features such as open, close, high, low, and volume.
